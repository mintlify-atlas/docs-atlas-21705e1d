    Quick Start
  </h2>
  <p className="text-base text-gray-600 dark:text-gray-400 mb-8">
    Get Mi Ruta Directa running in your development environment in just a few steps.
  </p>
  
  <Steps>
    <Step title="Clone the repository">
      Start by cloning the Mi Ruta Directa repository to your local machine.
      
      ```bash
      git clone https://github.com/Amurillo2005/Mi-Ruta-Directa
      cd Mi-Ruta-Directa
      ```
    </Step>
    
    <Step title="Install dependencies">
      Install all required packages using npm.
      
      ```bash
      npm install
      ```
      
      This will install React, Leaflet, Tailwind CSS, and all other dependencies needed to run the application.
    </Step>
    
    <Step title="Start the development server">
      Launch the development server to see the app in action.
      
      ```bash
      npm run dev
      ```
      
      Open your browser and navigate to the URL shown in the terminal (typically `http://localhost:5173`). You'll see the interactive map interface ready to use.
    </Step>
    
    <Step title="Try it out">
      Enter an origin address, destination address, and city name in the form fields. Click **Buscar** to calculate and display the route on the map.
      
      <Note>
        The app uses your browser's geolocation API to center the map on your current location when you first load the page.
      </Note>
    </Step>
  </Steps>
</div>
