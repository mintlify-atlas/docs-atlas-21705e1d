## Overview

The map display functionality is powered by **Leaflet**, the leading open-source JavaScript mapping library, integrated with React through **React Leaflet**. The `MapaFondo` component serves as the core map container, rendering base tiles, markers, routes, and interactive controls.

## Map Controls

The component uses two custom map controls imported from `MapaControles.tsx`:

### ObtenerUbicacionUsuario Control

Automatically centers the map on the user's location from `MapaFondo.tsx:38`:

```tsx src/components/MapaFondo.tsx
<ObtenerUbicacionUsuario position={posicionArray} />
```

Implementation from `MapaControles.tsx:4-14`:

```tsx src/components/MapaControles.tsx
export const ObtenerUbicacionUsuario = ({ position }: { position: [number, number] | null }) => {
  const map = useMap();
  
  useEffect(() => {
    if (position) {
      map.flyTo(position, 15);
    }
  }, [position, map]);
  
  return null;
}
```

- Uses `map.flyTo()` for smooth animated panning
- Zooms to level 15 (street view)
- Triggers when user position becomes available

### EnfocarRuta Control

Automatically fits the map bounds to show the entire route from `MapaFondo.tsx:50`:

```tsx src/components/MapaFondo.tsx
<EnfocarRuta coordenadas={datos.coordenadas} />
```

Implementation from `MapaControles.tsx:18-27`:

```tsx src/components/MapaControles.tsx
export const EnfocarRuta = ({ coordenadas }: { coordenadas: [number, number][] }) => {
  const map = useMap();
  
  useEffect(() => {
    if (coordenadas.length > 0) {
      map.fitBounds(coordenadas, { padding: [50, 50] });
    }
  }, [coordenadas, map])

  return null;
}
```

- Uses `map.fitBounds()` to calculate optimal zoom and center
- Adds 50px padding on all sides for comfortable viewing
- Updates whenever route coordinates change

<Info>
  These controls return `null` because they don't render any visible elementsâ€”they only manipulate the map imperatively using Leaflet's API through the `useMap()` hook.
</Info>

- All map elements are children of `MapContainer`

## useMap Hook

React Leaflet provides the `useMap()` hook for accessing the Leaflet map instance:

```tsx
import { useMap } from "react-leaflet";

const map = useMap();
```

Common map methods:

| Method | Description | Example |
|--------|-------------|----------|
| `flyTo(latlng, zoom)` | Animate to location | `map.flyTo([10, -74], 15)` |
| `fitBounds(bounds, options)` | Fit to bounds | `map.fitBounds(coords, {padding: [50, 50]})` |
| `setView(latlng, zoom)` | Jump to location | `map.setView([10, -74], 13)` |
| `getCenter()` | Get current center | `const center = map.getCenter()` |
| `getZoom()` | Get current zoom | `const zoom = map.getZoom()` |

<Note>
  The `useMap()` hook can only be called from components that are children of `MapContainer`. This is why `ObtenerUbicacionUsuario` and `EnfocarRuta` are separate components.
</Note>

## Performance Considerations

<Accordion title="Tile Loading">
  OpenStreetMap tiles load on-demand as users pan and zoom. On slow connections, tiles may appear with a delay. Consider showing a loading indicator or using a tile cache.
</Accordion>

<Accordion title="Re-rendering">
  The `MapContainer` should not be re-rendered unnecessarily. React Leaflet handles updates to child components (Markers, Polylines) efficiently, but recreating the entire map is expensive.
</Accordion>

<Accordion title="Memory Usage">
  Large polylines with thousands of coordinate points can impact performance. Consider simplifying routes using algorithms like Douglas-Peucker if needed.
</Accordion>
